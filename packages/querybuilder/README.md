# effect-qb

`effect-qb` is the typed SQL querybuilder package in this workspace.

## Install

```sh
bun add effect-qb
```

## Entry points

- `effect-qb/postgres`
- `effect-qb/mysql`
- `effect-qb/postgres/metadata`

`effect-qb/postgres/metadata` exposes normalized table and enum metadata helpers used by `effect-db`.
